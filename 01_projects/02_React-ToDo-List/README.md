## React ToDo App
This app is dedicated for my wife, who for some reason doesn't like the built in `notes` app of her iPhone.

The app also serves practicing purposes en route to become a better **`React`** *developer.*

## Tech
This app uses:
- separate components
- useEffect hook
- some basic CSS
